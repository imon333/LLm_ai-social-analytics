# Overview of Problems

### Problem 1: Social Media Bot Platform
Create a command-line social media platform populated by AI bots with distinct personalities.

**Key Features:**
- Bot personality definition using dictionaries
- Automated tweet generation about hobbies
- Maximum tweet length of 400 characters
- Timestamp and username display

**Optional Enhancements:**
- Command-line dashboard
- Bot commenting system
- User interaction capabilities

### Problem 2: Gender Bias in LLMs
Analyze gender stereotypes in LLM-generated content using profession-based stories.

**Implementation:**
- Story generation using llama_cpp
- Analysis of gender-associated pronouns
- Visualization of pronoun distributions
- Comparative analysis across professions

### Problem 3: ArXiv AI Publication Analysis
Analyze trends in AI paper publications on ArXiv.

**Tasks:**
- Yearly publication trend visualization
- Calculate total papers between 2002-2023
- Time series analysis

### Problem 4: ChatGPT Impact Analysis
Study the frequency of specific terms in research papers before and after ChatGPT's introduction.

**Focus Words:**
- delve
- intricate
- meticulous
- versatile
- pivotal

**Analysis Steps:**
- Text cleaning and preprocessing
- Frequency analysis
- Year-over-year comparison
- Letter distribution analysis
